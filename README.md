# Repositório de Programas em C

Este repositório contém programas de aprendizado em C/C++ durante meus estudos na faculdade, organizados por nível de complexidade: básico, intermediário e avançado.  

Cada pasta contém seus próprios exemplos e explicações detalhadas.  

Explore as pastas:
- `basico/`       : exercícios simples de lógica e loops  
- `intermediario/`: uso de arrays, matrizes e vetores  
- `avancado/`     : sistemas com menu, structs e manipulação de dados
